# Git commit conventions

Do not add `Co-Authored-By: Claude` (or any Claude/Anthropic co-author trailer) to commit messages. Commit messages should contain only the message itself, with no AI attribution trailer.

# Writing style

Do not use em dashes (`—`, U+2014) in any output: prose, code comments, READMEs, commit messages, GitHub descriptions, profile copy. Use a colon, comma, parenthesis, period (split the sentence), or " - " (hyphen with spaces) instead. Audit existing files before pushing if em dashes were added in earlier turns.
